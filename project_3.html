The Netflix Clone

I've built this basic clone project of the popular subscription-based streaming service Netflix.

  • In this project, you can watch selected movies/web series and it has a recommendation system too, which means it will recommend movies/web series based on what you watch.

  • You can upload videos/movies/series with their name, description, poster in Django Admin Panel with different categories/genres and they will show up on the website as Bootstrap Cards.

  • Then you can click the 'Watch' button to watch the video/movie/series.

  • It will recommend you similiar movies/series based on the categories/genres you watch.
